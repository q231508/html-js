Virtual Router [C]

A software-defined router implemented in C, capable of forwarding IP packets and responding to ARP and ICMP messages across a simulated network topology.
I implemented the core packet handling and forwarding logic, including Ethernet/IP/ICMP/ARP header parsing,
longest-prefix-match routing table lookups, TTL decrement and checksum recomputation, and ICMP error message generation for unreachable hosts, ports, and expired packets.
This includes the ARP subsystem, broadcasting ARP requests, caching replies, queuing packets pending MAC resolution, and sending ICMP host-unreachable messages after five failed attempts.

File System Simulator [C]

A UNIX like simulated file system implemented in C, built on top of a binary disk image with a superblock, inode table, and free block list.
I implemented the full suite of file system operations: mounting with six consistency checks,
file and directory creation, recursive deletion, block-level read and write, directory listing, cd navigation, and defragmentation.
This includes bit-level inode parsing for flags like in-use, directory, parent index, and file size,
as well as contiguous block allocation, free-space tracking, and compaction that shifts files toward the start of the disk to eliminate fragmentation.

Interactive Linux Shell [C]

A Linux shell implementation in C supporting command execution, background processes, I/O redirection, and single pipes.
I built the full shell loop including tokenization, fork/exec process management, a linked list job tracker,
and signal handling for SIGCHLD, SIGINT, and SIGTSTP to correctly manage foreground and background process lifecycles.
This includes built-in commands (cd, pwd, jobs, exit), stdin/stdout redirection with file descriptor duplication,
piped command execution across two child processes, and Ctrl+Z support that stops the foreground process without killing the shell.

Parallel Sorting by Regular Sampling [C]

A parallel sorting implementation of the PSRS (Parallel Sorting by Regular Sampling) algorithm in C using POSIX threads following the PSRS paper.
