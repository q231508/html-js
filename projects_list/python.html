Wordle Solver [Python]

A Wordle-solving project that uses a probabilistic model and heuristic search to guess five-letter words within six attempts.
I built the full system including game state representation, word bank generation, and a constraint-based search algorithm that narrows the candidate pool using green, yellow, and grey letter feedback.
This includes a bootstrapping training loop, a probability table model that reinforces successful guess paths, and a configurable exploit threshold that balances exploration and exploitation across 1,000 words.
Performance results are visualized through scatter and bar charts tracking solve rates and average guess counts across varying strategy configurations.

Retailer Review Organizer [Python]

A 3 person group project where we developed a MongoDB-backed product review database system for querying and analyzing large datasets of customer reviews.
We built the data usage pipeline and the full query interface consisting of batch loading of JSON review data into MongoDB and an interactive CLI for running aggregation queries.

Animated Taskbar Icons [Python]

A Windows system tray application that animates any GIF or image file as a live taskbar icon.
Built with Python using Pystray, Pillow, and Tkinter, the app processes each frame of a selected file, resizes it to fit the tray, and plays it back at 48 FPS on a dedicated thread.
This includes tray menu controls for run, stop, restart, and live file swapping, as well as frame-by-frame processing with progress feedback and error handling for invalid files.

Miscellaneous Cryptographic Tools [Python]

A suite of cryptographic implementations and attacks built across a security course.
This includes Shamir's Secret Sharing with polynomial-based share generation and Lagrange interpolation recovery,
a full Paillier cryptosystem supporting homomorphic evaluation of encrypted polynomials without exposing plaintext coefficients,
and Pollard's p-1 algorithm to factor an RSA modulus and recover the private key.
