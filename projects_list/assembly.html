RISC-V to ARM Translator[RISC-V Assembly]

A RISC-V to ARM binary translator written in RISC-V assembly, converting ALU and control flow instructions at the machine code level.
This includes bit-level decoding of R-type and I-type instructions, opcode and register mapping between ISAs, ARM immediate rotation field computation,
and a two-pass pipeline that first translates all instructions then resolves conditional branch and unconditional jump offsets by walking a RISC-V-to-ARM address table.
