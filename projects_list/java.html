Android Event Lottery Application [Java] [Android Studio]

A community event lottery Android application built with Java and Firebase where organizers post events and entrants join waiting lists to be fairly selected through a randomized draw.
This includes QR code scanning to view and join events, Firebase-backed real-time data for attendee lists and notifications, geolocation verification, and role-based access for entrants, organizers, and administrators.
Organizers can manage waiting lists, draw and redraw participants, send targeted notifications, and track enrollment status, while administrators can moderate content across the platform.
