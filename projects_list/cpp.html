Text Based Idle Game [C++]

A terminal-based incremental game built in C++ where players invest in various writing resources to reach $1,000,000.
I built the full game loop, UI rendering system, and upgrade mechanics,
including a double-buffered frame system using a string stream to eliminate terminal flicker on screen updates.
The game contains animated ASCII progress bars, a dynamic buy mode system (1x, 10x, 100x, MAX), exponential upgrade cost scaling,
production speed milestones that double output every 25 levels, and SFML audio integration for music and sound effects.
