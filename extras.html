HTML Archive

Welcome to the extras page!

One of my favorite features of HTML is how easy it is to build and share programs.
However impractical it may be, a single HTML is all you need for a wide array of ideas.

Here you can try out a growing list of HTML-JS projects I've been working on.
I've provided a brief description for each project, as well as a link to the code on github.

Feel free to return for new projects and improved indexing as I create more html-js projects.


Coordinate Tracker

Inspired by pointerpointer.com this is a simple cursor tracker. The webpage tracks cursor postion across four quadrants,
logging the cursor's X and Y position relative to the center of the page. Additionally, a reactive color gradient is included within each quadrant.
[Try it out!] [github]

Drawing Tool

Building off of the coordinate tracker, this is a simple drawing tool.
Written in vanilla HTML, CSS, JS, this project contains an export function, clearing the canvas,
color picking, brush resizing, erasing, and cursor tracking. A pixel editor will be created at a later date.
[Try it out!] [github]

Music Player

A standalone music player. Chosed between to upload .mp3 or .wav files one at a time or multiple at once.
Using JS, the collection of songs grows dynamically with each upload generating an associated entry on the page.
Songs can be looped, deleted, and pause when another track is playing.
[Try it out!] [github]
